Cara Menggunakan Alat Analisis Sentimen Komunitas Kami

1. Unggah Data Anda

Mulailah dengan mengunggah file .xlsx yang berisi data keluhan komunitas selama satu tahun. Pastikan file Anda mencakup teks keluhan dan tanggal.

  • Klik tombol "Unggah" di bilah navigasi
  • Pilih file .xlsx Anda dari komputer
  • Klik "Kirim" untuk memulai analisis

2. Lihat Hasil Analisis Sentimen

Setelah data Anda diproses, Anda akan melihat gambaran umum hasil analisis sentimen. Ini termasuk:

  • Skor sentimen keseluruhan
  • Distribusi sentimen negatif dan netral
  • Tren sentimen dari waktu ke waktu

3. Jelajahi Wawasan Detail

Selami lebih dalam data Anda dengan visualisasi interaktif kami:

  • Wordcloud yang menunjukkan istilah yang sering digunakan
  • Diagram batang topik keluhan teratas
  • Tren sentimen berdasarkan kategori atau periode waktu

4. Buat dan Ekspor Laporan

Buat laporan yang disesuaikan berdasarkan analisis Anda:

  • Pilih data dan visualisasi yang ingin Anda sertakan
  • Pilih format yang Anda inginkan (PDF atau Excel)
  • Unduh dan bagikan laporan Anda

Siap untuk Memulai?

Mulailah menganalisis data sentimen komunitas Anda sekarang dan dapatkan wawasan berharga!
