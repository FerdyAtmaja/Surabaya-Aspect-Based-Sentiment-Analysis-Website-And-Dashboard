Text Analysis Tool

How to Use

Petunjuk Input Text
  1. ketik atau salin teks yang ingin dianalisis ke dalam "Text Input" box.
  2. Pastikan teks inputan jelas untuk hasil terbaik.
  3. Klik tombol "Analisis Teks" untuk memproses inputan.
Petunjuk Unggah File
  1. Download templatenya di sini: Unduh Template
  2. Buka template menggunakan Microsoft Excel atau spreadsheet
  3. Isikan data Anda dengan format:
    • Kolom tanggal: Gunakan format YYYY-MM-DD HH:MM:SS
    • Kolom keluhan: Masukkan teks keluhan
  4. Simpan file
  5. Unggah data melalui tombol "Choose File".
  6. Klik "Analisis File" Untuk memproses data anda.
